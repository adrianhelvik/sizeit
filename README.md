# sizeit

A utility library for determining the complete size of the
contents within a folder.

## Usage
```bash
npx sizeit [directory]
```

```bash
npx sizeit

98.6kb   node_modules
22.5kb   .git
12.0kb   .README.md.swp
12.0kb   .index.js.swp
 2.1kb   yarn.lock
 1.6kb   index.js
195 b    package.json
 31 b    .gitignore
```
