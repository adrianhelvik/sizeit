# sizeit

A utility library for determining the complete size of the
contents within a folder.

## Usage
```bash
npx sizeit [directory]
```

```bash
npx sizeit

98.6KiB   node_modules
35.0KiB   .git
12.0KiB   .index.js.swp
 4.0KiB   .README.md.swp
 2.1KiB   yarn.lock
 1.8KiB   index.js
 327B     README.md
 222B     package.json
  31B     .gitignore
```
